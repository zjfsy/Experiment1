# Experiment1
## 功能概述:  
a. 当用户在命令行输入参数时，参数不为-f用参数中的字符串生成QrCode并在控制台中打印  
b. 当用户在命令行输入参数且参数为-f时，从-f后面的参数中读取文件路径，根据文件中的信息生成有二维码的bmp位图  
c. 限制输入的待处理的字符串长度在2048个字符以内
## 项目特色：  
从控制台传入参数，既能根据参数在控制台中直接生成QrCode，也能从文件中读取并将QrCode生成为位图文件
